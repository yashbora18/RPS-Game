# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

* 🪨 Rock, 📄 Paper, ✂️ Scissor gameplay
* 🤖 Computer generates random moves
* 📊 Score tracking system
* 🔊 Sound effects (Win / Lose / Tie)
* 🔄 Reset button
* 🎨 Attractive UI with animations

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
project/
 ├── index.html
 ├── style.css
 ├── script.js
 ├── win.mp3
 ├── lose.mp3
 └── tie.mp3
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start playing 🎮
